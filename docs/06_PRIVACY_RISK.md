## 1. 문서 목적

이 문서는 헤어샵 AI 상담 리포트 MVP에서 발생할 수 있는 개인정보, 얼굴 사진, 얼굴 분석 결과, AI 스타일 이미지, 상담 리포트, 클레임 리스크를 사전에 정리하기 위한 문서이다.

본 문서는 개발, 운영, 매장 도입, 고객 상담, 클레임 대응 시 기준 문서로 사용한다.

단, 본 문서는 내부 리스크 관리 초안이며, 실제 서비스 출시 전 개인정보보호법, 표시·광고 관련 법령, 소비자보호 관련 법령에 대한 법률 검토가 필요하다.

---

## 2. 기본 원칙

- 고객 이름과 전화번호는 기본 수집하지 않는다.
- 고객 식별이 필요한 경우에도 별도 동의를 받은 경우에만 최소한으로 수집한다.
- 얼굴 사진은 헤어 상담 목적 외 사용하지 않는다.
- 얼굴 사진과 얼굴 분석 결과는 민감한 개인정보로 보고 보수적으로 관리한다.
- 저장 동의가 없으면 상담 종료 후 원본 사진, 분석 결과, 추천 결과, 리포트를 삭제한다.
- 리포트 저장, 발송, 재방문 상담 보관은 각각 별도 선택 동의로 분리한다.
- 마케팅 활용 동의는 MVP 범위에서 제외한다.
- AI 분석 결과와 AI 스타일 이미지는 실제 시술 결과를 보장하지 않는다.
- 최종 시술 판단은 담당 디자이너가 고객의 모발 상태를 직접 확인한 후 진행한다.
- 고객에게 불리하거나 오해를 줄 수 있는 표현은 사용하지 않는다.
- 고객이 삭제를 요청하면 저장 동의 여부와 관계없이 즉시 삭제 절차를 진행한다.

## 3. MVP 개인정보 처리 범위

### 3.1 기본적으로 수집하지 않는 정보

MVP에서는 다음 정보를 기본 수집하지 않는다.

- 고객 이름
- 고객 전화번호
- 고객 생년월일
- 고객 주소
- 고객 SNS 계정
- 고객 회원 계정
- 결제 정보
- 신분증 정보
- 건강 정보
- 민감한 두피·질환 정보
- 보호자 정보

### 3.2 일시적으로 처리하는 정보

상담 진행을 위해 다음 정보는 일시적으로 처리될 수 있다.

| 항목               | 처리 목적                       | 저장 여부                          |
| ------------------ | ------------------------------- | ---------------------------------- |
| 얼굴 사진          | 얼굴형 분석 및 상담 리포트 생성 | 저장 동의 없으면 삭제              |
| 얼굴 랜드마크 좌표 | 얼굴 비율 및 형태 분석          | 저장 동의 없으면 삭제              |
| 얼굴 분석 결과     | 상담 참고 자료 생성             | 저장 동의 없으면 삭제              |
| 고객 선택 메뉴     | 추천 조건 계산                  | 세션 종료 후 삭제 또는 익명 통계화 |
| 고객 고민 선택값   | 추천 조건 계산                  | 세션 종료 후 삭제 또는 익명 통계화 |
| 추천 메뉴 결과     | 상담 리포트 생성                | 저장 동의 없으면 삭제              |
| 상담 리포트        | 디자이너 상담 보조              | 선택 동의 시에만 저장              |

### 3.3 저장 가능한 정보

고객이 명시적으로 저장에 동의한 경우에만 다음 정보를 저장할 수 있다.

| 항목           | 저장 조건        | 비고                              |
| -------------- | ---------------- | --------------------------------- |
| 상담 리포트    | 리포트 저장 동의 | 원본 사진 제외 권장               |
| 추천 메뉴 결과 | 리포트 저장 동의 | 매장 재상담용                     |
| 고객 선택값    | 리포트 저장 동의 | 메뉴, 고민, 원하는 스타일         |
| 분석 요약값    | 리포트 저장 동의 | 얼굴형, 추천 사유 등              |
| 발송 이력      | 발송 동의        | 카카오톡/메일 발송 시             |
| 삭제 로그      | 필수 보관        | 원본 데이터 없이 삭제 사실만 보관 |

### 3.4 원칙적으로 저장하지 않는 정보

다음 정보는 MVP에서 저장하지 않는 것을 원칙으로 한다.

- 원본 얼굴 사진
- 촬영 전후 전체 이미지
- 고객 신원 확인 정보
- 고객 연락처
- 주민등록번호 등 고유식별정보
- 건강 상태 또는 질병 정보
- 두피 질환 진단 정보
- 의학적 판단 정보
- 결제 정보
- 위치 정보

## 4. 동의 항목

## 4.1 필수 동의

서비스 이용을 위해 필요한 최소 동의 항목이다.

- 얼굴 사진 촬영
- 얼굴 분석
- 상담 리포트 생성을 위한 일시적 처리
- AI 분석 결과가 상담 참고용이라는 점 확인
- 저장 동의가 없을 경우 상담 종료 후 삭제된다는 점 확인

### 필수 동의 화면 표시 예시

아래 내용을 확인하고 동의해야 AI 상담 리포트 생성이 가능합니다.

- 얼굴 사진을 촬영합니다.
- 촬영된 사진은 얼굴형과 비율 분석을 위해 일시적으로 처리됩니다.
- 분석 결과는 담당 디자이너의 상담 참고 자료로만 사용됩니다.
- AI 분석 결과와 스타일 이미지는 실제 시술 결과를 보장하지 않습니다.
- 저장 동의가 없는 경우 상담 종료 후 사진과 분석 결과는 삭제됩니다.

[필수] 얼굴 사진 촬영 및 분석에 동의합니다.
[필수] AI 결과가 실제 시술 결과를 보장하지 않는 상담 참고 자료임을 확인했습니다.

---

## 4.2 선택 동의

고객 선택에 따라 별도로 받을 수 있는 동의 항목이다.

- 리포트 저장
- 카카오톡 발송
- 이메일 발송
- 다음 방문 상담을 위한 보관
- 매장 내부 상담 품질 개선을 위한 익명 통계 활용

MVP에서는 마케팅 활용 동의는 제외한다.

### 선택 동의 화면 표시 예시

아래 항목은 선택 사항입니다. 동의하지 않아도 상담 리포트 확인은 가능합니다.

[선택] 상담 리포트 저장에 동의합니다.
[선택] 상담 리포트 발송에 동의합니다.
[선택] 다음 방문 상담을 위한 리포트 보관에 동의합니다.
[선택] 개인을 식별할 수 없는 익명 통계 활용에 동의합니다.

---

## 4.3 MVP에서 제외할 동의 항목

MVP에서는 다음 동의 항목을 사용하지 않는다.

- 마케팅 문자 발송 동의
- 광고성 카카오톡 발송 동의
- 제휴사 제공 동의
- 외부 광고 플랫폼 연동 동의
- 고객 사진을 홍보 콘텐츠로 사용하는 동의
- 고객 얼굴을 AI 학습 데이터로 사용하는 동의

---

## 5. 필수 동의 문구 초안

본 프로그램은 고객님의 헤어 상담을 돕기 위해 얼굴 사진을 촬영하고, 얼굴형과 비율 등 일부 특징을 분석합니다.

촬영된 사진과 분석 결과는 담당 디자이너의 상담 참고 자료를 생성하기 위해 사용되며, 실제 시술 결과를 보장하지 않습니다.

저장 동의가 없는 경우 상담 종료 후 원본 사진, 분석 결과, 추천 결과, 상담 리포트는 삭제됩니다.

위 내용을 확인했으며, 얼굴 사진 촬영 및 분석에 동의합니다.

---

## 6. AI 결과 고지 문구

본 AI 분석 및 스타일 이미지는 고객 상담을 돕기 위한 참고 자료입니다.

실제 시술 결과는 다음 요소에 따라 달라질 수 있습니다.

- 현재 모발 상태
- 모발 손상도
- 기존 펌, 염색, 탈색 이력
- 모량
- 모질
- 두상
- 얼굴형 인식 오차
- 약제 반응
- 시술 시간
- 홈케어 방식
- 고객의 손질 습관
- 디자이너의 최종 판단

AI가 제안하는 스타일, 이미지, 메뉴 추천은 확정된 시술 결과가 아니며, 최종 시술 방향은 담당 디자이너와 상담 후 결정됩니다.

---

## 7. 저장/삭제 안내 문구

고객님의 사진과 분석 결과는 저장 동의가 없는 경우 상담 종료 후 삭제됩니다.

리포트 저장, 발송 또는 다음 방문 상담을 위한 보관을 원하시는 경우 별도 동의가 필요합니다.

저장에 동의하더라도 고객 요청 시 언제든지 삭제할 수 있습니다.

원본 얼굴 사진은 서비스 운영상 필요한 경우가 아니라면 저장하지 않는 것을 원칙으로 합니다.

---

## 8. 삭제 정책

## 8.1 저장 동의 없음

고객이 저장에 동의하지 않은 경우 상담 종료 후 다음 데이터를 삭제한다.

- 원본 얼굴 사진
- 얼굴 랜드마크 데이터
- 얼굴 분석 결과
- 추천 결과
- 상담 리포트
- 임시 캐시 데이터
- 브라우저 또는 로컬 DB의 세션 데이터

단, 다음 항목은 삭제 이력 확인을 위해 보관할 수 있다.

- 삭제 일시
- 삭제 대상 데이터 유형
- 삭제 처리 결과
- 삭제 처리 기기 ID
- 삭제 처리 담당자 또는 시스템 로그
- 고객을 직접 식별할 수 없는 세션 ID

삭제 로그에는 원본 사진, 얼굴 분석값, 고객 식별 정보가 포함되어서는 안 된다.

---

## 8.2 저장 동의 있음

고객이 저장에 동의한 경우 다음 기준에 따라 보관한다.

- 고객이 동의한 기간 동안 상담 리포트를 보관한다.
- 보관 기간은 매장 정책에 따라 별도 입력한다.
- 원본 얼굴 사진은 기본적으로 저장하지 않는 방향을 권장한다.
- 재방문 상담에 필요한 경우에도 원본 사진 대신 요약 리포트 중심으로 저장한다.
- 고객 요청 시 즉시 삭제한다.
- 보관 기간이 만료되면 자동 삭제한다.

### 권장 보관 데이터

- 상담 일시
- 선택한 시술 메뉴
- 선택한 고객 고민
- 추천 메뉴
- 추천 사유
- 디자이너 상담 메모
- AI 결과 고지 확인 여부
- 동의 이력
- 삭제 요청 이력

### 비권장 보관 데이터

- 원본 얼굴 사진
- 얼굴 원본 이미지
- 고해상도 촬영 이미지
- 불필요한 고객 식별 정보
- 건강·질환 관련 정보
- 외부 플랫폼 발송용 연락처의 장기 보관

---

## 8.3 삭제 요청 발생 시 처리 절차

고객이 삭제를 요청하면 다음 절차를 따른다.

1. 고객 요청 접수
2. 저장된 리포트 존재 여부 확인
3. 원본 사진 저장 여부 확인
4. 분석 결과 저장 여부 확인
5. 추천 결과 저장 여부 확인
6. 발송 이력 확인
7. 삭제 처리
8. 삭제 로그 기록
9. 고객에게 삭제 완료 안내

### 삭제 완료 안내 문구

고객님의 요청에 따라 저장된 상담 리포트 및 관련 분석 결과가 삭제되었습니다.

삭제 완료 후에는 해당 상담 리포트를 복구할 수 없습니다.

## 9. 클레임 리스크

| 리스크                       | 발생 상황                                                   | 대응                                               |
| ---------------------------- | ----------------------------------------------------------- | -------------------------------------------------- |
| AI 이미지와 실제 결과가 다름 | 고객이 AI 이미지와 동일한 결과를 기대한 경우                | 모든 결과 화면에 참고용 고지 표시                  |
| 얼굴형 분석 오류             | 얼굴형, 비율, 이미지 분석이 부정확한 경우                   | 분석 결과를 확정 판단이 아닌 상담 보조 자료로 표시 |
| 펌 결과 차이                 | 모발 손상도, 모질, 기존 시술 이력에 따라 결과가 다른 경우   | 모발 상태에 따른 차이 고지                         |
| 염색 결과 차이               | 기존 염색, 탈색, 모발 색소에 따라 색이 다르게 나오는 경우   | 실제 컬러는 상담 후 결정된다고 고지                |
| 클리닉 효과 과장             | 고객이 손상 복구 또는 치료 효과를 기대한 경우               | 복구, 치료, 완전 회복 표현 금지                    |
| 고객 사진 보관 불안          | 사진 저장 여부에 대한 고객 불신                             | 기본 미저장 구조와 삭제 안내 표시                  |
| 동의 없는 저장 논란          | 고객이 저장에 동의하지 않았는데 데이터가 남아 있는 경우     | 저장 동의 없는 세션 자동 삭제                      |
| 발송 동의 논란               | 리포트를 카카오톡 또는 메일로 보냈으나 동의가 불명확한 경우 | 발송 동의 체크박스 분리                            |
| 재방문 보관 논란             | 다음 방문 상담용으로 리포트를 보관했으나 고객이 몰랐던 경우 | 재방문 보관 동의 분리                              |
| 추천 메뉴 과잉 판매 오해     | 고객이 불필요한 시술을 권유받았다고 느낀 경우               | 추천 사유와 최종 판단 주체를 명확히 표시           |
| 디자이너 책임 전가           | AI 추천 때문에 디자이너 판단이 약화되는 경우                | 최종 판단은 담당 디자이너가 한다고 명시            |

## 10. 금지 표현

서비스 내 화면, 상담 리포트, 광고 문구, 디자이너 상담 스크립트에서 다음 표현은 사용하지 않는다.

### 10.1 결과 보장 표현

- 실제 결과 예측
- 실제 시술 결과와 동일
- 90% 이상 동일 구현
- 실패 없는 스타일
- 무조건 어울리는 스타일
- 시술 후 모습 그대로 재현
- 고객님에게 가장 완벽한 스타일
- AI가 결과를 보장합니다
- 이 스타일 그대로 나옵니다

### 10.2 분석 정확도 과장 표현

- 완벽한 얼굴형 분석
- 가장 정확한 얼굴 분석
- 100% 정확한 추천
- AI가 얼굴형을 완벽히 판단
- 실패 없는 추천
- 전문가보다 정확한 분석
- 얼굴형을 과학적으로 완벽 진단

### 10.3 의료·치료 오인 표현

- 복구 클리닉
- 손상 완전 회복
- 모발 치료
- 두피 치료
- 탈모 개선
- 의학적 효과
- 손상모 완전 재생
- 모발 건강을 치료합니다

### 10.4 과도한 판매 표현

- 반드시 해야 하는 시술
- 안 하면 손상됩니다
- 이 메뉴가 정답입니다
- 지금 하지 않으면 늦습니다
- 고객님은 이 스타일밖에 안 됩니다
- AI가 선택한 필수 시술

## 11. 권장 표현

서비스 내 화면, 상담 리포트, 디자이너 상담 스크립트에서는 다음 표현을 사용한다.

### 11.1 AI 결과 안내 표현

- 상담 참고용
- 스타일 방향성
- 추천 후보
- 실제 결과와 차이 가능
- 담당 디자이너 상담 후 최종 결정
- 모발 상태에 따라 조정 필요
- 고객님의 현재 모발 상태를 확인한 뒤 결정 필요
- AI 분석 결과는 상담 보조 자료입니다
- 최종 시술 가능 여부는 디자이너가 판단합니다

### 11.2 추천 메뉴 안내 표현

- 현재 선택하신 고민을 기준으로 추천된 메뉴입니다
- 매장에 등록된 실제 시술 메뉴 중 추천된 항목입니다
- 고객님의 모발 상태에 따라 추천 메뉴가 변경될 수 있습니다
- 담당 디자이너 상담 후 최종 시술 메뉴가 결정됩니다
- 시술 전 충분한 상담이 필요합니다

### 11.3 클리닉 안내 표현

- 모발 컨디션 관리
- 손상 완화에 도움
- 윤기 관리
- 부드러운 결 관리
- 시술 전후 케어
- 모발 상태에 따라 체감 효과가 다를 수 있습니다

| 발송 선택 화면     | 발송을 위해 별도 동의 필요               | 체크박스         |

## 13. 상담 리포트 하단 고지 문구

본 리포트는 고객 상담을 돕기 위한 참고 자료입니다.

AI 분석 결과와 스타일 이미지는 실제 시술 결과를 보장하지 않으며, 실제 결과는 모발 상태, 기존 시술 이력, 손상도, 모질, 모량, 두상, 약제 반응, 손질 방식에 따라 달라질 수 있습니다.

최종 시술 방향은 담당 디자이너와 충분히 상담한 후 결정됩니다.

---

## 14. 디자이너 상담 시 안내 문구

AI가 추천한 결과는 고객님의 선택값과 얼굴 분석 정보를 바탕으로 만든 상담 참고 자료입니다.

실제 시술 가능 여부는 고객님의 현재 모발 상태, 손상도, 기존 시술 이력, 모질, 모량을 직접 확인한 후 결정해야 합니다.

따라서 리포트에 나온 이미지는 완성 결과를 보장하는 이미지가 아니라, 상담 방향을 정하기 위한 참고 이미지로 안내해야 합니다.

## 15. 관리자 입력 제한 정책

관리자가 시술 메뉴, 추천 조건, 상담 문구를 등록할 때 다음 제한을 적용한다.

### 15.1 메뉴명 제한

다음 표현이 포함된 메뉴명은 등록하지 않는다.

- 복구
- 치료
- 완전 회복
- 탈모 개선
- 100% 개선
- 실패 없음
- 무조건
- 보장

### 15.2 설명 문구 제한

관리자 입력 문구에는 다음 표현을 사용하지 않는다.

- 실제 결과를 보장합니다
- AI가 정확히 예측했습니다
- 이 스타일 그대로 구현됩니다
- 손상이 완전히 회복됩니다
- 모든 고객에게 효과가 있습니다
- 반드시 만족합니다

### 15.3 추천 조건 제한

추천 조건은 고객의 외모를 단정하거나 평가하는 방식으로 작성하지 않는다.

금지 예시:

- 얼굴이 커 보이는 고객
- 단점이 많은 얼굴형
- 못생겨 보이는 인상
- 나이 들어 보이는 얼굴
- 광대가 심한 얼굴
- 턱이 이상한 얼굴

권장 예시:

- 얼굴형 보완을 원하는 고객
- 부드러운 인상을 원하는 고객
- 볼륨감을 원하는 고객
- 얼굴 라인을 자연스럽게 연출하고 싶은 고객
- 손질이 쉬운 스타일을 원하는 고객

MVP 단계에서는 다음 보관 기준을 권장한다.

| 데이터         | 기본 정책                   | 보관 기간             |

| 추천 결과      | 저장 동의 없으면 삭제       | 상담 종료 즉시 삭제   |
| 상담 리포트    | 선택 동의 시 저장           | 매장 정책에 따라 설정 |

## 17. 외부 전송 및 제3자 처리 리스크

MVP에서 외부 API, 카카오톡 발송, 이메일 발송, 클라우드 저장소를 사용하는 경우 다음 사항을 검토한다.

- 외부 업체에 어떤 데이터가 전달되는지 확인한다.
- 원본 얼굴 사진이 외부 서버로 전송되는지 확인한다.
- 전송 데이터에 고객 식별 정보가 포함되는지 확인한다.
- 위탁 처리인지 제3자 제공인지 구분한다.
- 고객 동의 화면에 외부 전송 여부를 표시한다.
- 전송 후 외부 서버에 데이터가 남는지 확인한다.
- 외부 업체의 보관 기간과 삭제 정책을 확인한다.

MVP에서는 가능한 한 로컬 처리 우선 구조를 사용한다.

## 18. 로컬 저장 보안 원칙

태블릿 또는 매장 로컬 환경에 데이터를 저장하는 경우 다음 기준을 적용한다.

- 저장 동의 없는 데이터는 세션 종료 시 자동 삭제한다.
- 관리자 화면은 비밀번호 또는 PIN으로 보호한다.
- 상담 리포트 목록은 일반 고객 화면에서 접근할 수 없도록 한다.
- 태블릿 분실 시 데이터 노출을 최소화할 수 있도록 저장 데이터를 최소화한다.
- 원본 사진은 저장하지 않는 것을 기본값으로 한다.
- 로컬 DB에는 불필요한 개인정보를 저장하지 않는다.
- 삭제 로그에는 고객 얼굴 이미지나 분석값을 포함하지 않는다.
- 관리자 비밀번호는 평문으로 저장하지 않는다.
- 기기 교체 또는 매장 계약 종료 시 전체 데이터 삭제 절차를 제공한다.

## 19. 미성년자 이용 리스크

미성년 고객이 서비스를 이용할 가능성이 있으므로 다음 기준을 둔다.

- 만 14세 미만 고객의 개인정보 처리는 별도 법률 검토 전까지 MVP에서 제외한다.
- 만 14세 미만으로 보이는 고객에게는 보호자 동의 없이 사진 촬영 및 분석을 진행하지 않는다.
- 미성년자 고객의 리포트 저장 또는 발송 기능은 기본 비활성화한다.
- 매장 직원에게 미성년자 이용 시 별도 확인 절차를 안내한다.

## 20. 장애 및 오류 발생 시 대응

### 20.1 분석 오류

분석 오류가 발생한 경우 다음 문구를 표시한다.

얼굴 분석을 완료하지 못했습니다.
촬영 환경, 조명, 얼굴 방향, 카메라 접근 권한을 확인한 뒤 다시 시도해 주세요.

분석 오류가 반복될 경우 AI 분석 없이 담당 디자이너 상담으로 진행할 수 있습니다.

### 20.2 추천 오류

추천 결과가 생성되지 않은 경우 다음 문구를 표시한다.

추천 결과를 생성하지 못했습니다.
선택한 메뉴와 고민을 다시 확인해 주세요.

AI 추천 없이 담당 디자이너 상담으로 진행할 수 있습니다.

### 20.3 삭제 오류

삭제 처리에 실패한 경우 다음 절차를 따른다.

1. 고객에게 삭제 지연 사실 안내
2. 관리자 화면에 삭제 실패 로그 표시
3. 재삭제 버튼 제공
4. 삭제 완료 후 삭제 로그 기록
5. 필요 시 수동 삭제 절차 진행

## 21. 고객 문의 대응 기준

### 21.1 “사진이 저장되나요?”

기본적으로 저장되지 않습니다.
저장 동의를 하지 않으신 경우 상담 종료 후 사진과 분석 결과는 삭제됩니다.
리포트 저장 또는 발송을 원하시는 경우에만 별도 동의를 받고 처리합니다.

### 21.2 “AI 이미지처럼 똑같이 나오나요?”

아닙니다.
AI 이미지는 상담을 돕기 위한 참고 이미지입니다.
실제 결과는 고객님의 모발 상태, 기존 시술 이력, 손상도, 모질, 모량, 약제 반응, 손질 방식에 따라 달라질 수 있습니다.
최종 시술 방향은 담당 디자이너와 상담 후 결정됩니다.

### 21.3 “분석 결과가 틀린 것 같아요.”

AI 분석 결과는 확정 진단이 아니라 상담 참고 자료입니다.
촬영 각도, 조명, 얼굴 방향, 표정 등에 따라 분석 결과가 달라질 수 있습니다.
최종 판단은 담당 디자이너 상담을 통해 진행됩니다.

### 21.4 “리포트를 삭제하고 싶어요.”

요청하시면 저장된 상담 리포트와 관련 분석 결과를 삭제할 수 있습니다.
삭제 완료 후에는 해당 리포트를 복구할 수 없습니다.

### 21.5 “제 사진이 마케팅에 쓰이나요?”

MVP에서는 고객님의 얼굴 사진을 마케팅 용도로 사용하지 않습니다.
홍보 콘텐츠, 광고, 포트폴리오 등에 사용하려면 별도의 명확한 동의가 필요합니다.

## 24. 최종 운영 원칙

이 서비스는 고객의 헤어 상담을 돕는 보조 도구이다.

AI 분석 결과, 스타일 이미지, 추천 메뉴는 고객과 디자이너가 상담 방향을 정하기 위한 참고 자료이며, 실제 시술 결과를 보장하지 않는다.

고객의 얼굴 사진과 분석 결과는 필요한 범위에서만 처리하고, 저장 동의가 없는 경우 상담 종료 후 삭제한다.

서비스 운영자는 고객이 자신의 사진과 리포트가 어떻게 처리되는지 명확히 이해할 수 있도록 안내해야 한다.

개인정보 보호, 결과 고지, 삭제 정책, 클레임 방어 문구는 MVP 단계부터 반드시 포함한다.
