# 02_ux_flow.md

## 목적

헤어샵 태블릿 앱의 고객, 디자이너, 관리자 플로우를 정리하기 위한 프롬프트입니다.

## 프롬프트

너는 태블릿 기반 매장용 서비스 UX 설계자다.

헤어샵 AI 상담 리포트 프로그램의 사용자 플로우를 설계해줘.

사용자는 3종류다.

1. 고객
2. 디자이너
3. 매장 관리자

서비스 조건:

- 고객은 태블릿에서 시술 메뉴와 고민을 선택한다.
- 고객은 원하는 이미지와 손질 가능 정도를 선택한다.
- 디자이너는 개인정보 동의 확인 후 사진을 촬영한다.
- 촬영 화면에는 얼굴 가이드 프레임이 표시된다.
- 분석 중 화면에는 폴리곤 스캔 UI가 표시된다.
- 얼굴 분석 후 매장 실제 메뉴 기반 추천 리포트가 나온다.
- AI 이미지는 실제 결과 보장이 아니라 상담 참고용이다.
- 저장 동의가 없으면 상담 종료 후 삭제된다.
- 관리자는 매장 메뉴, 가격, 소요시간, 추천 조건을 등록한다.
- 전체 상담 플로우는 3분 이내를 목표로 한다.

## 출력 형식

1. 고객 플로우
2. 디자이너 플로우
3. 관리자 플로우
4. 각 단계별 필요한 화면
5. 각 단계별 입력 데이터
6. 각 단계별 출력 데이터
7. 각 단계별 저장해야 할 데이터
8. 개인정보 동의가 필요한 지점
9. 클레임 방어 고지가 필요한 지점
10. MVP에서 생략 가능한 단계
11. 태블릿 화면 UX 주의사항
12. 개발 우선순위
