## 제약 조건

- React + TypeScript + Vite 프로젝트다.
- 태블릿 웹앱 기준이다.
- 개인정보/삭제 흐름이 깨지면 안 된다.
- 임시방편보다 유지보수 가능한 수정이 필요하다.
- 고객 얼굴 사진이 외부로 전송되면 안 된다.
- 저장 동의가 없는 데이터는 삭제 가능해야 한다.

## 요청

1. 가능한 원인 후보를 우선순위대로 정리해줘.
2. 어떤 파일을 확인해야 하는지 알려줘.
3. 가장 가능성 높은 수정 방향을 제안해줘.
4. 수정 시 부작용이 생길 수 있는 부분을 알려줘.
5. 테스트해야 할 시나리오를 작성해줘.
6. 개인정보/삭제 흐름에 영향이 있는지 확인해줘.
