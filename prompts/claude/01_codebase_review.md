# 01_codebase_review.md

## 목적

현재 코드베이스 구조가 헤어샵 AI 상담 리포트 MVP에 적합한지 검토하기 위한 프롬프트입니다.

## 프롬프트

이 프로젝트는 헤어샵 태블릿용 AI 상담 리포트 MVP입니다.

목표:

- 고객이 시술 메뉴와 고민을 선택한다.
- 개인정보 동의 후 얼굴 촬영을 진행한다.
- 얼굴 분석 결과를 바탕으로 매장 실제 메뉴를 추천한다.
- 상담 리포트를 제공한다.
- 저장 동의가 없으면 고객 사진, 분석 결과, 리포트를 삭제한다.

현재 코드베이스와 docs 폴더를 검토해줘.

검토 기준:

1. 1인 개발자가 3개월 안에 MVP를 완성하기에 구조가 과하지 않은가
2. 기능별 폴더 구조가 명확한가
3. 고객 플로우, 관리자 플로우, 추천 엔진이 분리되어 있는가
4. 개인정보 동의/삭제 기능이 구조적으로 빠져 있지 않은가
5. 로컬 DB 확장이 가능한가
6. 나중에 PWA 또는 Capacitor로 전환하기 쉬운가
7. 테스트하기 쉬운 구조인가
8. 매장 메뉴 기반 추천 구조가 확장 가능한가
9. AI 헤어 합성처럼 MVP 범위를 벗어난 코드가 섞여 있지 않은가

## 출력 형식

1. 전체 평가
2. 좋은 점
3. 구조상 문제점
4. 당장 수정해야 할 항목
5. 나중에 수정해도 되는 항목
6. 추천 폴더 구조
7. 리팩토링 제안
8. 누락된 문서
9. 다음 개발 우선순위
