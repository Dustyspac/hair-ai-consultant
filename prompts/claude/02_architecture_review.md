# 02_architecture_review.md

## 목적

로컬 우선 태블릿 앱 구조가 적절한지 검토하기 위한 프롬프트입니다.

## 프롬프트

이 프로젝트의 목표는 헤어샵 태블릿에서 구동되는 로컬 우선 AI 상담 리포트 앱입니다.

예정 기술:

- React
- TypeScript
- Vite
- Tailwind CSS
- 로컬 DB
- MediaPipe Face Landmarker
- 추후 PWA 또는 Capacitor
- 선택적으로 서버 연동

핵심 요구사항:

- 고객 이름과 전화번호는 기본 수집하지 않는다.
- 얼굴 사진은 저장 동의가 없으면 삭제한다.
- 분석 결과와 리포트도 저장 동의가 없으면 삭제한다.
- 추천은 매장에 등록된 실제 메뉴와 연결되어야 한다.
- 고객 상담 플로우는 3분 이내여야 한다.
- 태블릿 가로 화면 기준이다.
- 관리자 설정에서 메뉴, 가격, 추천 조건을 등록해야 한다.
- AI 결과는 실제 시술 결과를 보장하지 않는다.

현재 아키텍처를 검토하고 개선안을 제시해줘.

## 출력 형식

1. 현재 구조 평가
2. 데이터 흐름 검토
3. 로컬 DB 구조 검토
4. 개인정보 삭제 흐름 검토
5. 추천 엔진 구조 검토
6. MediaPipe 연동 시 주의사항
7. PWA/Capacitor 전환 시 고려사항
8. 장기적으로 서버 연동이 필요한 지점
9. 현재 단계에서 과한 설계
10. 즉시 반영할 개선사항
11. MVP 이후 확장 방향
